# Setting Up Express Server

## Prerequisites
Make sure you have these installed:

Node.js (LTS version)

Git

## Run the server

1. Clone the repo
   ```bash
   git clone https://github.com/Mahdi-Al-Sabea/eCommerceAS_server.git
   cd eCommerceAS_server
   ```

2. Install dependencies
   ```bash
   npm install
   ```

   

4. Start running
   ```bash
   npm run dev
   ```

 The server will start running on port 4000 , for simplicity no database was used , all data are temporaly saved on memory
